# Øliv Ecommerce Website Clone

This is a modern frontend clone of the Øliv website, built using pure HTML and CSS.

##  Features

- Clean and minimal UI design
- Flexbox-based structure
- Multiple sections (Navbar, Cards, Work, Services, etc.)
- Smooth hover effects and transitions

##  Tech Stack

- HTML5
- CSS3 (Flexbox)

##  Project Structure

- Navbar section
- Hero section
- Services cards
- Work showcase
- Values section
- Final content layout (text + paragraphs)

##  What I Learned

- Flexbox layout deeply
- Spacing and alignment techniques
- Real-world UI cloning
- Responsive design basics
